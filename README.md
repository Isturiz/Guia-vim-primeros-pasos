### Iniciar vim desde la terminal<a name="1"></a>  

Vim: `vim`

Neovim: `nvim`

### Salir de Vim / Neovim

| Comando   | Acción |
| :-----:   | :----: | 
| `q`       | salir  | 
| `q!`      | forzar salida (sin guardar) |

### Moverse en el editor

| Comando   | Acción    |
| :-----:   | :----:    | 
| `h`       |izquierda  |
| `j`       |abajo      |
| `k`       |arriba     |
| `l`       |derecha    |

| Acción   | Comando    |
| :-----:   | :----:    | 
|        izquierda  |`h`|
|        abajo      |`j`|
|        arriba     |`k`|
|        derecha    |`l`|
