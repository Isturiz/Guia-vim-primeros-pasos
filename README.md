### Iniciar vim desde la terminal<a name="1"></a>  

Vim: `vim`

Neovim: `nvim`

### Salir de Vim / Neovim<a name="2"></a>  

| Comando   | Acción |
| :-----:   | :---- | 
| `q`       | salir  | 
| `q!`      | salir sin guardar |

### Moverse en el editor<a name="3"></a> 

| Comando   | Acción    |
| :-----:   | :----    | 
| `h`       |izquierda  |
| `j`       |abajo      |
| `k`       |arriba     |
| `l`       |derecha    |
| `w`       |siguiente  |
| `e`       |final palabra  |
| `l`       |anterior   |
